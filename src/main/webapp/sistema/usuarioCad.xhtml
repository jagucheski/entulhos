Usuários

Coordenador possui acesso a todas as funcionalidades do sistema e é o único com acesso irrestrito ao sistema.

Usuário Comum: O usuário comum possui acesso restrito ao sistema, dessa forma, pode apenas visualizar as informações cadastradas;

Usuário Administrativo: O usuário administrativo possui permissão para realizar tarefas administrativas, dessa forma, possui acesso ao cadastro de usuários, clientes, entrada e saida de cargas, etc;
